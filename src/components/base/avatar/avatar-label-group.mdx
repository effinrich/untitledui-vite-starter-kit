# AvatarLabelGroup

The `AvatarLabelGroup` component provides configurable sizes.

## Usage

### Basic Example

<Canvas of={AvatarLabelGroupStories.Default} />

```tsx
<AvatarLabelGroup size="md">
  Content
</AvatarLabelGroup>
```

### Sizes

Available in 4 sizes:

<Canvas of={AvatarLabelGroupStories.Sizes} />

```tsx
<AvatarLabelGroup size="sm">sm</AvatarLabelGroup>
<AvatarLabelGroup size="md">md</AvatarLabelGroup>
<AvatarLabelGroup size="lg">lg</AvatarLabelGroup>
<AvatarLabelGroup size="xl">xl</AvatarLabelGroup>
```

