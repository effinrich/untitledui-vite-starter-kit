# AvatarProfilePhoto

The `AvatarProfilePhoto` component provides configurable sizes.

## Usage

### Basic Example

<Canvas of={AvatarProfilePhotoStories.Default} />

```tsx
<AvatarProfilePhoto size="md">
  Content
</AvatarProfilePhoto>
```

### Sizes

Available in 3 sizes:

<Canvas of={AvatarProfilePhotoStories.Sizes} />

```tsx
<AvatarProfilePhoto size="sm">sm</AvatarProfilePhoto>
<AvatarProfilePhoto size="md">md</AvatarProfilePhoto>
<AvatarProfilePhoto size="lg">lg</AvatarProfilePhoto>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"sm" \| "md" \| "lg"` | `-` | - |
