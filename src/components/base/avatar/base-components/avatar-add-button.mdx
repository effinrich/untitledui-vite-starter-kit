# AvatarAddButton

The `AvatarAddButton` component provides configurable sizes.

## Usage

### Basic Example

<Canvas of={AvatarAddButtonStories.Default} />

```tsx
<AvatarAddButton size="sm">
  Content
</AvatarAddButton>
```

### Sizes

Available in 3 sizes:

<Canvas of={AvatarAddButtonStories.Sizes} />

```tsx
<AvatarAddButton size="xs">xs</AvatarAddButton>
<AvatarAddButton size="sm">sm</AvatarAddButton>
<AvatarAddButton size="md">md</AvatarAddButton>
```

## Props

| `className` | `string` | `-` | - |


## Accessibility

The `AvatarAddButton` component follows WAI-ARIA guidelines:

- Supports keyboard navigation
- Includes proper ARIA attributes
- Compatible with screen readers

### Keyboard Interactions

| Key | Description |

## Responsive Design

Use CSS media queries for responsive behavior:

```tsx

  size="xs"
  style={{ width: '100%' }}
  className="responsive-component"
>
  Responsive AvatarAddButton
</AvatarAddButton>

<style>
  @media (min-width: 768px) {
    .responsive-component {
      width: auto;
      /* Use sm size */
    }
  }
</style>
```

## Best Practices

5. **Performance**: Avoid creating new callback functions on each render
