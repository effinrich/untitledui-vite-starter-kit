# AvatarCompanyIcon

The `AvatarCompanyIcon` component provides configurable sizes.

## Usage

### Basic Example

<Canvas of={AvatarCompanyIconStories.Default} />

```tsx
<AvatarCompanyIcon size="sm">
  Content
</AvatarCompanyIcon>
```

### Sizes

Available in 6 sizes:

<Canvas of={AvatarCompanyIconStories.Sizes} />

```tsx
<AvatarCompanyIcon size="xs">xs</AvatarCompanyIcon>
<AvatarCompanyIcon size="sm">sm</AvatarCompanyIcon>
<AvatarCompanyIcon size="md">md</AvatarCompanyIcon>
<AvatarCompanyIcon size="lg">lg</AvatarCompanyIcon>
<AvatarCompanyIcon size="xl">xl</AvatarCompanyIcon>
<AvatarCompanyIcon size="2xl">2xl</AvatarCompanyIcon>
```

## Responsive Design

Use CSS media queries for responsive behavior:

```tsx
<AvatarCompanyIcon
  size="xs"
  style={{ width: '100%' }}
  className="responsive-component"
>
  Responsive AvatarCompanyIcon
</AvatarCompanyIcon>

<style>
  @media (min-width: 768px) {
    .responsive-component {
      width: auto;
      /* Use sm size */
    }
  }
</style>
```
