# AvatarOnlineIndicator

The `AvatarOnlineIndicator` component provides configurable sizes.

## Usage

### Basic Example

<Canvas of={AvatarOnlineIndicatorStories.Default} />

```tsx
<AvatarOnlineIndicator size="sm">
  Content
</AvatarOnlineIndicator>
```

```tsx
<style>
  @media (min-width: 768px) {
    .responsive-component {
      width: auto;
      /* Use sm size */
    }
  }
</style>
```
