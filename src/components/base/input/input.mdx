# Input

The `Input` component is a reusable UI element.

## Usage

### Basic Example

<Canvas of={InputStories.Default} />

```tsx
<Input>
  Content
</Input>
```

## Props

<ArgTypes of={InputStories} />

|------|------|---------|-------------|
| `hideRequiredIndicator` | `boolean` | `-` | - |


## Accessibility

The `Input` component follows WAI-ARIA guidelines:

- Supports keyboard navigation
- Includes proper ARIA attributes
- Compatible with screen readers

### Keyboard Interactions

| Key | Description |
|-----|-------------|
| `Tab` | Moves focus to the component |
| `Enter` | Activates the component |
| `Space` | Activates the component |

## Responsive Design

Use CSS media queries for responsive behavior:

```tsx
<Input
  
  style={{ width: '100%' }}
  className="responsive-component"
>
  Responsive Input
</Input>

<style>
  @media (min-width: 768px) {
    .responsive-component {
      width: auto;
      
    }
  }
</style>
```
