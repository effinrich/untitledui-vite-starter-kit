# Select

The `Select` component is a reusable UI element.

## Usage

### Basic Example

<Canvas of={SelectStories.Default} />

```tsx
<Select>
  Content
</Select>
```

## Props

<ArgTypes of={SelectStories} />

|------|------|---------|-------------|
| `items` | `SelectItemType[]` | `-` | - |

| `placeholderIcon` | `FC \| ReactNode` | `-` | - |
| `children` | `ReactNode \| ((item: SelectItemType) => ReactNode)` | `-` | - |


## Accessibility

The `Select` component follows WAI-ARIA guidelines:

- Supports keyboard navigation
- Includes proper ARIA attributes
- Compatible with screen readers

### Keyboard Interactions

| Key | Description |
|-----|-------------|
| `Tab` | Moves focus to the component |
| `Enter` | Activates the component |
| `Space` | Activates the component |

## Responsive Design

Use CSS media queries for responsive behavior:

```tsx
<Select
  
  style={{ width: '100%' }}
  className="responsive-component"
>
  Responsive Select
</Select>

<style>
  @media (min-width: 768px) {
    .responsive-component {
      width: auto;
      
    }
  }
</style>
```
