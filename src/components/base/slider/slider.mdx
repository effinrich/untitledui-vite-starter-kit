# Slider

The `Slider` component is a reusable UI element.

## Usage

### Basic Example

<Canvas of={SliderStories.Default} />

```tsx

  Content
</Slider>
```

## Accessibility

The `Slider` component follows WAI-ARIA guidelines:

- Supports keyboard navigation
- Includes proper ARIA attributes
- Compatible with screen readers

### Keyboard Interactions

| Key | Description |
|-----|-------------|
| `Tab` | Moves focus to the component |
| `Enter` | Activates the component |
| `Space` | Activates the component |

## Responsive Design

Use CSS media queries for responsive behavior:

```tsx
<Slider
  
  style={{ width: '100%' }}
  className="responsive-component"
>
  Responsive Slider
</Slider>

<style>
  @media (min-width: 768px) {
    .responsive-component {
      width: auto;
      
    }
  }
</style>
```
