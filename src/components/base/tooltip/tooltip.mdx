# Tooltip

The `Tooltip` component is a reusable UI element.

## Usage

### Basic Example

<Canvas of={TooltipStories.Default} />

```tsx
<Tooltip>
  Content
</Tooltip>
```

## Props

<ArgTypes of={TooltipStories} />

| `arrow` | `boolean` | `-` | - |
| `delay` | `number` | `-` | - |


## Accessibility

The `Tooltip` component follows WAI-ARIA guidelines:

- Supports keyboard navigation
- Includes proper ARIA attributes
- Compatible with screen readers

### Keyboard Interactions

| Key | Description |
|-----|-------------|
| `Tab` | Moves focus to the component |
| `Enter` | Activates the component |
| `Space` | Activates the component |

## Responsive Design

Use CSS media queries for responsive behavior:

```tsx
<Tooltip
  
  style={{ width: '100%' }}
  className="responsive-component"
>
  Responsive Tooltip
</Tooltip>

<style>
  @media (min-width: 768px) {
    .responsive-component {
      width: auto;
      
    }
  }
</style>
```
